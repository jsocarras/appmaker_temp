# Coin Toss Simulator User Manual

## Introduction

The Coin Toss Simulator is a simple application built using Streamlit, a Python library for creating interactive web apps. This simulator allows users to flip a virtual coin and displays whether it lands on heads or tails.

## Installation

To use the Coin Toss Simulator, you need to install the required dependencies. Follow the steps below to set up the environment:

1. Make sure you have Python installed on your machine. You can download Python from the official website: [https://www.python.org/downloads/](https://www.python.org/downloads/)

2. Open a terminal or command prompt and navigate to the project directory.

3. Create a virtual environment (optional but recommended) by running the following command:

   ```
   python -m venv myenv
   ```

4. Activate the virtual environment:

   - For Windows:

     ```
     myenv\Scripts\activate
     ```

   - For macOS/Linux:

     ```
     source myenv/bin/activate
     ```

5. Install the required dependencies by running the following command:

   ```
   pip install -r requirements.txt
   ```

## Usage

Once you have installed the dependencies, you can run the Coin Toss Simulator by following these steps:

1. Open a terminal or command prompt and navigate to the project directory.

2. Activate the virtual environment (if you created one).

3. Run the following command to start the Streamlit app:

   ```
   streamlit run main.py
   ```

4. A web browser window should open automatically, displaying the Coin Toss Simulator.

5. Click the "Flip Coin" button to flip the virtual coin.

6. Wait for a moment, and the result (either "Heads" or "Tails") will be displayed on the screen.

7. You can continue flipping the coin by clicking the "Flip Coin" button again.

## Conclusion

Congratulations! You have successfully installed and used the Coin Toss Simulator. Have fun flipping the virtual coin and enjoy the randomness of the results!
