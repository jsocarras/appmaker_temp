# Random Number Generator App

The Random Number Generator app is a simple Streamlit application that generates a random number when the user presses a button. This manual will guide you through the installation process and explain how to use the app.

## Installation

To use the Random Number Generator app, you need to install the required dependencies. Follow the steps below:

1. Make sure you have Python installed on your system. If not, you can download it from the official Python website: [https://www.python.org/downloads/](https://www.python.org/downloads/)

2. Open a terminal or command prompt and navigate to the directory where you have the `requirements.txt` file.

3. Run the following command to install the dependencies:

   ```
   pip install -r requirements.txt
   ```

   This command will install the necessary packages, including Streamlit.

## Usage

Once you have installed the dependencies, you can run the Random Number Generator app. Follow the steps below:

1. Open a terminal or command prompt and navigate to the directory where you have the `main.py` file.

2. Run the following command to start the app:

   ```
   streamlit run main.py
   ```

3. The app will start running, and you will see a URL in the terminal. Open that URL in your web browser.

4. The Random Number Generator app will be displayed in your browser. It will have a title "Random Number Generator" and a button "Generate Random Number".

5. Click on the "Generate Random Number" button to generate a random number. The generated number will be displayed below the button.

6. You can click the button again to generate a new random number.

7. The app will also automatically update the random number every 5 seconds.

That's it! You can now use the Random Number Generator app to generate random numbers by pressing the button or let it update automatically every 5 seconds.
